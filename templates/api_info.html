dontabandon.me provides a simple read-only API (thanks to Django Rest Framework). The browseable api can be found here. Here is a list of all possible requests to the API:
  • api/authors – authors in alphabetical order
  • api/authors/votes – authors ordered by number of votes
  • api/authors/projects – authors ordered by number of projects
  • api/authors/1 – single author page (replace 1 with author id)
  • api/languages – languages in alphabetical order
  • api/languages/votes – languages ordered by number of votes
  • api/languages/projects – languages ordered by number of projects
  • api/languages/1 – single language page (replace 1 with language id)
  • api/projects – projects in alphabetical order
  • api/projects/latest – projects ordered by add date
  • api/projects/votes – projects ordered by number of votes
  • api/projects/1 – single project page (replace 1 with project id)
  • api/reasons – reasons in alphabetical order
  • api/reasons/votes – reasons ordered by number of votes
  • api/reasons/projects – reasons ordered by number of projects
  • api/reasons/1 – single reason page (replace 1 with reason id)
  • api/tags – tags in alphabetical order
  • api/tags/votes – tags ordered by number of votes
  • api/tags/projects – tags ordered by number of projects
  • api/tags/1 – single tag page (replace 1 with tag id)
